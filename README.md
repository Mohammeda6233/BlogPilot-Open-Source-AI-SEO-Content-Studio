## 📊 Modules Explained

BlogPilot breaks down complex SEO into simple steps. 

### Keyword Research
Use the discovery module to find terms your audience types into search engines. The tool suggests topics based on search volume and difficulty scores.

### Content Creation
The editor module helps you write articles. You can set a target keyword and receive suggestions for headers, keywords, and tone.

### Technical Audit
The crawl module looks at your website links. It finds broken pages, missing descriptions, and structural issues that might hide your content from search engines.

### Schema Markup
The markup tool builds hidden tags for your pages. These tags tell Google exactly what your content is about, such as recipes, reviews, or product info.
