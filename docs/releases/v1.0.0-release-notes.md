## 🚀 Quick start

```bash
git clone https://github.com/IamRamgarhia/blogpilot-ai
cd blogpilot-ai

# macOS / Linux
./install.sh && ./start.sh

# Windows
.\install.ps1
.\start.ps1
```

Open http://localhost:3000.

See [README.md](https://github.com/IamRamgarhia/blogpilot-ai#readme) for full docs · [DEPLOY.md](https://github.com/IamRamgarhia/blogpilot-ai/blob/main/DEPLOY.md) for Vercel + Docker · [CHANGELOG.md](https://github.com/IamRamgarhia/blogpilot-ai/blob/main/CHANGELOG.md) for the full feature list.
